# ball-in UI 設計預覽

> **此為設計預覽頁面，非正式產品。** 正式頁面將依實際功能開發進度與需求調整。

## 線上預覽

**https://jhihwei.github.io/SportsLotto-issues-archive/**

## 功能

- **4 個樣本頁面**：房間列表、預測房間、排行榜、邀請分享
- **Dark / Light 主題切換**：右上角按鈕（PC）
- **PC / Mobile 切換**：右上角切換按鈕（PC 版可見）
- **Banner 輪播**：自動播放，支援手動切換
- **公告區**：活動、公告、更新三種標籤分類
- **像素風 UI 素材**：使用 itch.io 免費素材包的錦旗、像素框、道具槽、按鈕等元件

## 素材來源

使用的像素 UI 素材來自 itch.io 免費資源包（Free版），僅供設計預覽用途。
